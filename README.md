Trooperfall
===========

Casual game prototype

Building
--------

Building is OS dependent. 
For all OS:

    > cd $COCOS_HOME
    > git submodule add git@github.com:jjay/trooperfall
    > cd trooperfall
    > ./bin/spriti.py animations

Ubuntu
------

Install eclipse cdt
Export project from dir $COCOS_HOME/trooperfall/linux
